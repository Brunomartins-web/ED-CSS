# Valores e unidades de medidas 

Cada propriedade possui valores
`property: value`

`<Data-type>`

> Estudo constante a fim de entender as propeiedades e seus valores

## Display: Block

1. Ocupa toda a linha como um `bloco`
2. `width` e `heigth` são aplicados
3. `padding`, `margin`, `border` funcionam por comleto.

>Elementos
`<div>, <main>, <header>, <selection>, <p>, <h....>`

## Display: inline

1.Ocupa apenas o espaço do conteudo do elemento.
2.Elementos poderão ficar `em linha`.
3.`width` e `heigth` não se aplicam.
3. Aplicamos apenas os valores horizontais de `margin`, `padding` e `border`.

>Elementos
`<a>, <span>, <strong>, <em>`
